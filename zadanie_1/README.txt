Tworzenie nowych rewizji oraz sprawdzanie stanu projektu
1. Utwórz nowe repozytorium o następującej strukturze:
	- A
		- file1.txt
		- file2.txt
	- B
		- file1.txt
		- file2.txt
2. Dodaj wszystkie pliki z folderu A do poczekalni
3. Stwórz nową rewizję z wiadomością "init"
4. Zmodyfikuj plik A/file1.txt
5. Dodaj do poczekalni wszystkie pliki
6. Sprawdź zmiany wprowadzone w plikach znajdujących się w poczekalni
7. Zatwierdź wprowadzone zmiany. Nie korzystaj z parametru –m
8. Dodaj nowy plik w folderze B. Zatwierdź zmiany z wykorzystaniem parametru --amend. Zamieść w wiadomości informację o nowym pliku

!! Po każdej komendzie git add oraz git commit sprawdź stan repozytorium !!
