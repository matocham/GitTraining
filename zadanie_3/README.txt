Ignorowanie plików

1. Stwórz nowy projekt za pomocą wybranego środowiska IDE, rozwiązujący prosty problem, np. kalkulator
2. Zainicjuj nowe repozytorium oraz stwórz plik .gitignore, który będzie ignorował pliki specyficzne dla danego IDE. Skorzystaj ze strony https://gitignore.io 
3. W czasie pracy nad programem stwórz co najmniej 3 nowe rewizje.
4. Po skończeniu programu dodaj znacznik o nazwie v1.0 oraz dowolnym komentarzu
5. Sprawdź informacje o ostatnim znaczniku korzystając z poleceń git log --tags oraz git show <tag_name>
6. Korzystając z polecenia git rebase -i scal ze sobą rewizje HEAD~1 oraz HEAD~2. Dostosuj odpowiednio komentarz rewizji.
7. Sprawdź rezultat poleceniem git log
